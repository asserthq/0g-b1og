# 0g-b1og
Learning Laravel 

## Clone
```
git clone https://github.com/asserthq/0g-b1og.git
cd 0g-b1og
```

## Set up
- Generate depedencies in vendor folder
    ```
    composer install
    ```
- Rename `.env.example` to `.env`

- Generate app key
    ```
    php artisan key:generate
    ```
- Configure `.env`

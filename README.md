# 0g-b1og
Learning Laravel 

## Clone
```sh
git clone https://github.com/asserthq/0g-b1og.git
cd 0g-b1og
```

## Set up
- Generate depedencies in vendor folder
    ```sh
    composer install
    ```
- Rename `.env.example` to `.env`

- Generate app key
    ```sh
    php artisan key:generate
    ```
- Configure `.env`

## Run
```sh
php artisan serve
```
